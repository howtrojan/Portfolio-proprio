Conhecimentos

↑

HTML5

HTML5 é a quinta versão da linguagem para web, melhorando semântica com novas tags, suporte nativo a áudio/vídeo, e recursos avançados, sendo crucial para o desenvolvimento web moderno.

CSS3

CSS3 é a terceira versão das folhas de estilo para web, aprimorando design com transições, animações, sombras e layout responsivo, sendo essencial para estilos avançados em páginas web modernas.

BOOTSTRAP

Bootstrap é um framework front-end, simplificando o desenvolvimento web com um sistema de grid responsivo, componentes prontos e estilos predefinidos, agilizando a criação de interfaces modernas e adaptáveis.

JAVASCRIPT

JavaScript é uma linguagem de programação client-side essencial para desenvolvimento web, permitindo interatividade, manipulação de elementos na página e comunicação assíncrona, sendo fundamental para experiências dinâmicas e responsivas na web moderna.

REACT

React é uma biblioteca JavaScript para construir interfaces de usuário interativas, baseada em componentes reutilizáveis. Desenvolvida pelo Facebook, simplifica a criação de aplicativos web eficientes, proporcionando uma atualização eficiente e uma gestão simplificada do estado da aplicação.
